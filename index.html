Science Fair 2026

An Investigation into Automated Web Deployment & QR Integration

Abstract

This project explores the efficiency of utilizing Python-based automation to bridge the gap between physical media and digital cloud hosting. By deploying a static site via GitHub Pages, we demonstrate a low-latency method for information dissemination.

Methodology

  • Scripting with Python to handle data encoding.
  • Cloud synchronization via Git version control.
  • Optimization of UI/UX using CSS Grid and Flexbox.

Results

The experiment resulted in a 100% success rate for QR-to-URL redirection, proving that cloud-native hosting is the most stable environment for science fair presentations.
